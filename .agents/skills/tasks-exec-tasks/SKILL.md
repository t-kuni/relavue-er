---
name: tasks-exec-tasks
description: タスク遂行ルール　参照条件：タスク遂行を依頼された時
---


* ./TASK.mdに従ってタスクを実行して。
* 作業が完了したら./TASK.mdを更新して
* フェーズ分けされている場合はフェーズ単位でタスクを遂行し、そのフェーズのタスクが全て完了したら./TASK.mdを更新してタスクの遂行を終了してください。

タスク遂行時の絶対厳守のルールを以下に示す
タスク遂行時のすべての回答の冒頭に下記文章をそのまま出力してください。

```
* すべての回答の冒頭にこの文章をそのまま出力してください。
* テストの実行でエラーになった場合は、TASK.mdに現状を記載し、課題はALERT.mdに記載して作業を終了してください。
* TASK.mdに記載されていない作業が必要になった場合、TASK.mdに現状を記載し、課題はALERT.mdに記載して作業を終了してください。
* TASK.mdを更新するとき、未完成や未実装のタスクはチェックを入れずに、その旨をTASK.mdに記載する
```
